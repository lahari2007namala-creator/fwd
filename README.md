## About the Project

**NEXTGEN** is a feature-rich, single-page educational platform designed for engineering students pursuing **CSE (Computer Science Engineering)** and **CSIT (Computer Science & Information Technology)** courses across all 4 years of study.

It combines structured academic content, an AI/ML learning hub, an in-browser code editor, and a smart topic search — all in a sleek dark-themed UI.

---

## ✨ Features

### 📚 Subject Library
- Covers **CSE & CSIT** branches, **Year 1 to Year 4**
- 40+ subjects including Data Structures, Algorithms, DBMS, OS, CN, ML, Cloud Computing, and more
- Each subject has topic-wise notes with **concept explanations**, **logic building**, and **Q&A sections**

### 🤖 AI Hub
- Dedicated section for **Artificial Intelligence & Machine Learning**
- Topics: ML Foundations, Deep Learning, NLP, Computer Vision, Reinforcement Learning
- Sidebar navigation with detailed subtopic content

### 💻 Program Practice (Code Editor)
- In-browser code editor supporting **Python, Java, C, C++**
- **Run code** directly in the browser (Python & Java via simulation)
- **Live Web Preview** for HTML/CSS/JS code
- Integrated **VS Code** and **GitHub** deployment shortcuts

### 🔍 Smart Search
- Rich search results with **algorithm steps**, **multi-language code**, **complexity analysis**, and **interview Q&A**
- Keyword-based intelligent matching
- Topics covered: Binary Search, Linked Lists, Stacks, Sorting, Pointers, and more

### 🎨 UI/UX
- Futuristic dark theme with neon accents
- Fully **responsive design** (CSS Grid + Flexbox)
- Smooth hover animations and transitions
- Custom scrollbar and grid background

##  How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/lahari2007namala-creator/fwd.git
   ```

2. **Navigate to the project folder:**
   ```bash
   cd fwd
   ```

3. **Open in browser:**
   ```bash
   open index.html
   # or simply double-click index.html
   ```

No dependencies or installations required — it's a pure HTML/CSS/JS project.

---

##  Deployment

This project is deployed using **GitHub Pages**.

To deploy your own copy:
1. Go to your repo → **Settings** → **Pages**
2. Under *Source*, select branch `main` and folder `/ (root)`
3. Click **Save** — your site will be live at `https://<your-username>.github.io/fwd`
